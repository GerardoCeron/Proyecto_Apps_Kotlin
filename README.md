# DocTranslate 

## Descripción

El objetivo principal es facilitar el acceso a la información en múltiples idiomas y mejorar la accesibilidad para usuarios que requieran contenido en formato de audio. 
DocTranslate es una aplicación móvil Android desarrollada en Kotlin que le permite al usuario traducir documentos entre distintos idiomas, convertir archivos a diferentes 
formatos y generar audio a partir del texto traducido utilizando la API de ElevenLabs (por implementar), y con la API de MyMemory Translation API
